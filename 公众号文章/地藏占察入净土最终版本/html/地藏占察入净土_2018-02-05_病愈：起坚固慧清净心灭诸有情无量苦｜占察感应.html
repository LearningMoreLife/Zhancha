{  给宝宝占病  }

作者：幸福妙音


缘起：

（2016年11月12日）我的女儿去装修房子了，把小外孙女送给我带几天，孩子白天精神状态很好，晚上睡觉不踏实，翻来翻去折腾，搞得我也没睡好。这孩子一向很好带的，我以为是非人众生干扰，就一直念诵观音菩萨心咒。到了第二天，孩子时不时地咳嗽，还流鼻涕，我才知道她有点感冒了，但估计不会有啥大事的，没往心里去。到了下午，想想还是按占病流程给宝宝占一下。


占察如下：

1

简轨问：弟子妙音的小外孙女某某感冒咳嗽病因？祈请地藏菩萨以观所患轮相145-147为弟子开示。


57把：52+58+35=145 观所患大不调

2

问：弟子妙音想知道治疗此病的难易程度，祈请地藏菩萨以轮相148、149为弟子开示。


22把：44+56+49=149 观所患难疗治

3

问：弟子妙音想知道念诵地藏菩萨圣号功德回向，是否对治感冒咳嗽，祈请地藏菩萨以轮相154、158为弟子开示。


占了好多把没出轮相，天看着要晚了，就没有占。

我跟群里一个师兄说了占察宝宝感冒咳嗽的事，他让我用专求轮相103试试，我说好吧，明早一起来就求103轮相。


就这么又到了夜里，宝宝睡觉又开始折腾。我看看凌晨三点多了，摸摸她的额头，感觉孩子发烧了，身上也很热，才知道孩子昨天夜里折腾，应该也是发烧，我疏忽大意了。这时我有点担心了，家里啥药也没有，我起来坐着一直念诵地藏菩萨圣号，念到五点多起床，洗漱完做早课，全轨后求103轮相，占察如下：


4

弟子妙音祈请地藏菩萨赐予103轮相，加持弟子的小外孙女某某感冒咳嗽发烧得除愈。


28把：27+44+32=103 求离病得除愈


轮相出来后，我算了又算害怕加错了，当确定无疑时，我感恩的泪水止不住地往下流，宝宝没事了。我跪拜大慈大悲大愿大力的地藏菩萨，感激涕零，感恩南无大愿大力地藏菩萨摩诃萨！！！


宝宝的爷爷奶奶爸爸疼心特别重，都不在身边，要是知道宝宝生病会着急叫送医院，真的不想让孩子去医院打针吃药，孩子三岁了，还没去医院吊过水呢。


专求103轮相出来后，宝宝就没有发烧症状了，虽然还时不时地咳嗽一阵，但不是很厉害，流鼻涕也不厉害，每天还正常去幼儿园。我就这么每天观察孩子的病情，一天努力多念诵地藏菩萨圣号回向给宝宝。我看着宝宝病一天比一天轻了，那活泼可爱的样子，真是感到这孩子太有福报了，生活在学佛人的家中，特别是学习占察法门的佛弟子家中，从小就接触到佛教，在她幼小的心灵深处就种下了金刚种子，待今生缘分成熟时，极乐世界的七宝莲池又将多一朵美丽的七宝莲花。


现在孩子已经好了，轮相完美应验！从我给孩子占察病因开始，专求轮相103求到后，到病好过程只有一个星期时间，要是以前，最起码十天左右，不但吃药还要受好多罪，不知家人要操多少心呢。


感恩我的家人都是菩萨来成就我的，自从学占察以来，两个多月了，为家人和自己看病一个接一个，没有花过一分钱吃药打针，久练久熟，让我忙得不亦乐乎。


占察法门太好了，真是不可思议的法宝，这一生能够进占察群，跟着智坤师父如法学习占察法门，是我一生中之大幸！


感恩南无大愿大力地藏菩萨摩诃萨慈悲加持！！！

感恩智坤师父慈悲！

感恩师兄们的帮助！



{  腹部疼痛得除愈  }

作者：佛门弟子

2017.9.27

缘起：

末学于9月13日吃完晚饭后感觉右侧腹部有点疼，用手一按有一疼点，两日后加重，走路拜忏都疼，于是末学于15日上午去XX二院看医生。给我问诊的是一名年轻医生，他说我可能是运动时肌肉拉伤。回去用热敷即可，不见好再来。于是我回去后就用毛巾热敷了几分钟。感觉好点，但第二天又痛，且睡觉也痛了。于是末学于17日自己占了下病情。结果如下：

1

简轨：弟子问（用手指向疼的地方给菩萨看）这里痛的病因，请菩萨以观所患145-147轮相慈悲开示。


27把：46+54+45=145 观所患大不调

按流程走，于是继续问菩萨：

2

弟子问治疗该病的难易程度，请菩萨以观所患148、149轮相慈悲开示。


菩萨给了148观所患可疗治轮相

3

又问：弟子去XX二院看该病，请菩萨以所向医153、157轮相慈悲开示。


32把：54+42+57=153 观所患向医堪能治


得到这轮相后我心里有了点底。但我心想还是专家靠谱，便于18日下午又去XX二院挂一专家号。专家一看觉得严重，怀疑末学是肾或肠子内部的病，先做彩超查肾。肾没问题，又让我做肠镜，做肠镜前先要做心电和验血后才能预约，这些检查我从18日下午到19日下午才算做完。这期间右侧腹部痛点没轻反倒有点重了。肠镜检查预约到了20日上午。


19日下午回来后我看了下17日所占的轮相不免有些疑惑。仔细想了下所向医轮相的设问，末学只是问去XX二院看病并没有问先前去的还是随后去的呀！于是末学直接问菩萨。如下：

4

弟子问先前去XX二院找年轻医生看病，他说弟子的病是肌肉拉伤让弟子热敷。该治疗方案对弟子的病是否对治，请菩萨以154、158轮相慈悲开示。


20把：54+43+57=154 观所疗是对治


此轮相一出我不再怀疑，年轻医生的诊断是正确的！虽然腹部痛点还在，但我还是取消了做肠镜的预约。因为我对菩萨深信不疑。


应验：

在我取消预约的第二天后，腹部的痛点痛感立减，第三天痛感全部消失，只稍有不适感，第五天不适感也不复存在。身体恢复如初。


经过这次看病和占病的经历，我明白依靠菩萨的重要性。依靠菩萨不能嘴上说说，要落在实处、深处，乃至于我们的生命中。而这种依靠是要对菩萨有一定的坚固信，此信心从何而来？从占察中来！


从专家诊断到做彩超看肾，做肠镜，当时末学感觉病得很重了，幸亏在关键时刻末学选择的是依靠菩萨而不是专家，不然从身心到财力乃至精神上不知还要遭受多少折磨！


通过自己亲身的经历使末学对菩萨的信心又上了一新的台阶，只要你信，菩萨就在你身边！


至诚顶礼感恩地藏菩萨摩诃萨，感恩智坤师父领我进入占察法门，感恩众师兄无私的帮助。




占察净土一相逢，生活修行再不同。

两大愿王亲摄取，直登九莲出牢笼。
